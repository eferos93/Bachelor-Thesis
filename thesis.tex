\section{Functional Requirements}
Functional requirements are statements of services the system should provide, particularly how it should react to particular inputs.
The Language Server Protocol has six features that can be implemented: 
\begin{itemize}
	\item Code completion
	\item Hover information
	\item Jump to definition
	\item Workspace symbols
	\item Find references
	\item Diagnostics
\end{itemize}
From those features we extracted the following functional requirements, in ascending order of importance.
